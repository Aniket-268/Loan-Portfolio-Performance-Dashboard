# Credit Risk & Loan Performance Dashboard

##  Overview
This project analyzes bank loan data to understand customer credit behavior, loan approval patterns, and default risk. The goal is to help financial institutions make better lending decisions using data-driven insights.

The dashboard visualizes key loan performance indicators such as approval rates, default trends, customer segments, and repayment behavior.

---

##  Dataset
The dataset includes information about loan applicants and approved loans, such as:
- Customer income
- Credit history
- Loan amount
- Loan status
- Employment details
- Loan term
- Default and repayment status

---

##  What This Project Does
- Cleans and prepares bank loan data  
- Analyzes customer risk profiles  
- Identifies patterns in loan approvals and defaults  
- Segments customers based on financial behavior  
- Tracks loan portfolio performance  
- Presents insights through an interactive dashboard  

---

##  Key Insights
The dashboard helps answer:
- Which customer segments have higher default risk?
- How income and credit history affect loan approvals
- Which loan types perform better
- Trends in loan repayment over time
- Risk distribution across customer groups

Banks rely on accurate risk analysis to reduce losses and improve lending quality. This project demonstrates how data analytics and dashboards can support smarter financial decision-making.
